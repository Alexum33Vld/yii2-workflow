Interface raoul2000\workflow\source\IWorkflowSource

All Classes | Methods
Implemented byraoul2000\workflow\source\file\WorkflowFileSource

Interface that must be implemented by WorkflowSource components.

This interface defines basic methods aimed to provide status, workflow and transitions to the SimpleStatusBehavior.
